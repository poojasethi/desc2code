from fractions import gcd
n = int(raw_input())
m = map(int, raw_input().split())
r = map(int, raw_input().split())
a = [[] for _ in xrange(720720 + 100)]
for i in xrange(n):
    a[r[i]].append(m[i])
for i in xrange(720720):
    for j in a[i]:
        a[i+j].append(j)
    a[i] = 1 if a[i] else 0
print sum(a[:720720]) / 720720.0
